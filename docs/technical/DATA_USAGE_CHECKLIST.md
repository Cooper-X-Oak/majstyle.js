## 附录：字段详细说明

### 默听率计算

```
默听率 = 和牌率 - 立直率 - 副露率
```

**说明**: 默听率反映玩家不鸣牌、不立直而直接和牌的频率。高默听率说明玩家擅长隐蔽听牌，进攻意图不明显。

### 净打点效率计算

```
净打点效率 = 打点效率 - 铳点损失
```

**说明**: 净打点效率综合反映玩家的进攻和防守能力。正值说明得点大于失点，实力强劲。

### 进攻意愿计算

```
进攻意愿 = 立直率 + 副露率
```

**说明**: 立直和副露都是主动进攻的表现，两者之和反映玩家的进攻意愿。

### 进攻效率计算

```
进攻效率 = (和牌率 / 进攻意愿) × 100
```

**说明**: 进攻效率反映玩家进攻的成功率。高效率说明玩家能有效转化进攻意愿为和牌。
