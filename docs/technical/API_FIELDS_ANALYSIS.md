## 🆕 新发现的高价值字段（推荐使用）

### 1. 和了形式分析

| 字段名 | 类型 | 含义 | 价值评估 | 用途 |
|--------|------|------|----------|------|
| 自摸率 | number | 自摸率 | ⭐⭐⭐ 高 | 判断是否偏好自摸，影响进攻策略 |
| 默听率 | number | 默听率（不立直不副露） | ⭐⭐⭐ 高 | 判断是否喜欢默听，影响危险度评估 |
| 立直和了 | number | 立直和了次数 | ⭐⭐ 中 | 辅助分析立直效率 |
| 副露和了 | number | 副露和了次数 | ⭐⭐ 中 | 辅助分析副露效率 |
| 默听和了 | number | 默听和了次数 | ⭐⭐ 中 | 辅助分析默听效率 |

**分析价值**:
- 可以区分"立直型"、"副露型"、"默听型"玩家
- 默听率高的玩家更危险（不易察觉）
- 自摸率高说明进攻质量好

| 字段名 | 类型 | 含义 | 价值评估 | 用途 |
|--------|------|------|----------|------|

## 🎯 建议系统增强方案

基于新发现的字段，可以实现以下增强：

### 1. 风格分类更精细

**当前**: 9种主称号
**增强**: 可以区分更多子类型

- **立直型玩家**
  - 早立型（立直巡目 < 8）
  - 晚立型（立直巡目 > 12）
  - 高效立直型（立直收支 > 3000）
  - 赌博立直型（振听立直率 > 5%）

- **副露型玩家**
  - 高效副露型（副露后和牌率 > 35%）
  - 失误副露型（副露后放铳率 > 18%）

- **默听型玩家**
  - 默听率 > 15% 的玩家

### 2. 危险度评估更准确

**新增维度**:
- 净打点效率（综合实力）
- 默听率（隐蔽性）
- 被炸率（大牌防守）
- 平均铳点（放铳损失）

**新公式**:
```javascript
危险度 = (
  净打点效率权重 * 净打点效率分数 +
  默听率权重 * 默听率分数 +
  立直质量权重 * 立直质量分数 +
  副露质量权重 * 副露质量分数
) / 总权重
```

### 3. 建议更具体

**示例**:

**当前建议**:
- "对手进攻意愿强，建议抢先进攻"

**增强后建议**:
- "对手净打点效率 +775（高于平均），默听率 11.8%（中等），立直收支 +3116（盈利），综合实力强"
- "对手平均和了巡数 12.6（中速），立直巡目 9.8（中等），不算特别快"
- "对手被炸率 7.9%（正常），但平均铳点 5364（偏低），放铳损失可控"
- "**建议**: 正面对抗风险中等，可以正常打法，注意对手可能默听"
