# 牌谱屋 API 数据结构档案

> 创建时间: 2026-03-14
> 目的: 记录牌谱屋 API 的完整数据结构，为后续功能开发提供数据基础

## 使用说明

1. 打开 `tools/api-explorer.html` 工具
2. 输入一个真实的雀魂账号 ID
3. 点击"开始探索"按钮
4. 将完整的 JSON 响应复制到本文档

## player_stats API

### 端点

```
GET https://5-data.amae-koromo.com/api/v2/pl4/player_stats/{accountId}/{startTime}/{endTime}?mode=12.9&tag=492541
```

### 参数说明

- `accountId`: 雀魂账号 ID
- `startTime`: 开始时间戳（毫秒）
- `endTime`: 结束时间戳（毫秒）
- `mode`: 游戏模式（12.9 = 金玉四麻）
- `tag`: 标签（492541 = 段位场）

### 返回字段

| 字段名        | 类型     | 含义   | 当前使用 | 价值评估 | 备注       |
| ---------- | ------ | ---- | ---- | ---- | -------- |
| level.id   | number | 段位ID | ✅    | 高    | 用于匹配基准数据 |
| level.name | string | 段位名称 | ✅    | 高    | 显示用      |
| ...        | ...    | ...  | ...  | ...  | 待补充      |

### 完整响应示例

```json
{
  "待补充": "请使用 api-explorer.html 工具获取实际数据"
}
```

## 数据价值评估标准

### 高价值

- 直接影响对局策略的指标
- 可用于风险评估的数据
- 能够区分玩家风格的特征

### 中价值

- 辅助分析的指标
- 用于趋势判断的数据
- 补充信息

### 低价值

- 纯展示性数据
- 重复或冗余信息
- 难以解读的指标
