# 安装指南

本文档提供详细的安装步骤和常见问题解决方案。

## 前置要求

- 支持的浏览器：Chrome、Edge、Firefox、Opera
- 需要安装用户脚本管理器（推荐 Tampermonkey）

## 步骤 1：安装 Tampermonkey

### Chrome / Edge

1. 访问 [Tampermonkey 官网](https://www.tampermonkey.net/)
2. 点击对应浏览器的下载链接
3. 在扩展商店页面点击"添加到 Chrome/Edge"
4. 确认安装

### Firefox

1. 访问 [Firefox 插件商店](https://addons.mozilla.org/firefox/addon/tampermonkey/)
2. 点击"添加到 Firefox"
3. 确认安装

### 验证安装

安装成功后，浏览器工具栏会出现 Tampermonkey 图标（一个黑色的小方块）。

## 步骤 2：安装脚本

### 方法一：通过 Tampermonkey 管理面板

1. 点击浏览器工具栏的 Tampermonkey 图标
2. 选择"管理面板"
3. 点击左侧的"+"号（新建脚本）
4. 删除编辑器中的默认内容
5. 打开项目根目录的 `雀魂金玉四麻风格分析助手-v2.0.0.user.js` 文件
6. 复制全部内容（Ctrl+A, Ctrl+C）
7. 粘贴到 Tampermonkey 编辑器（Ctrl+V）
8. 保存（Ctrl+S 或点击文件图标）

### 方法二：直接拖拽安装（推荐）

1. 下载 `雀魂金玉四麻风格分析助手-v2.0.0.user.js` 文件到本地
2. 将文件拖拽到浏览器窗口
3. Tampermonkey 会自动识别并弹出安装确认页面
4. 点击"安装"按钮

### 验证安装

1. 点击 Tampermonkey 图标 → 管理面板
2. 在脚本列表中应该能看到"雀魂金玉四麻风格分析助手"
3. 确认脚本状态为"已启用"（开关为绿色）

## 步骤 3：使用脚本

1. 打开雀魂游戏网站：
   - 国服：https://game.maj-soul.com
   - 国际服：https://game.maj-soul.net

2. 登录游戏账号

3. 进入金之间或玉之间的四人麻将对局

4. 等待 2-3 秒，脚本会自动分析并显示玩家风格信息

## 常见问题

### Q: 安装后没有任何显示？

**可能原因和解决方案**：

1. **脚本未启用**
   - 打开 Tampermonkey 管理面板
   - 确认脚本开关为绿色（已启用）

2. **不在支持的对局中**
   - 脚本仅支持金之间、玉之间的四人麻将
   - 不支持三麻、友人战、比赛场

3. **浏览器控制台有错误**
   - 按 F12 打开开发者工具
   - 查看 Console 标签页是否有错误信息
   - 如有错误，请提交 Issue 反馈

### Q: 显示"无牌谱数据"？

这是正常情况，表示该玩家：
- 在金之间/玉之间的对局数不足 50 局
- 是新注册的账号
- 没有在牌谱屋收录的数据

### Q: 脚本加载很慢？

- 脚本需要为每个玩家请求两次 API（基础数据 + 扩展数据）
- 网络延迟会影响加载速度
- 通常 3-5 秒内完成加载

### Q: 如何更新脚本？

1. 打开 Tampermonkey 管理面板
2. 找到"雀魂金玉四麻风格分析助手"
3. 点击脚本名称进入编辑页面
4. 复制新版本的全部内容
5. 粘贴替换原有内容
6. 保存（Ctrl+S）

### Q: 如何卸载脚本？

1. 打开 Tampermonkey 管理面板
2. 找到"雀魂金玉四麻风格分析助手"
3. 点击右侧的垃圾桶图标
4. 确认删除

## 权限说明

脚本需要以下权限：

- `GM_xmlhttpRequest`：用于请求牌谱屋 API
- `unsafeWindow`：用于访问游戏对象获取玩家 ID
- `@match`：仅在雀魂游戏页面运行
- `@connect`：允许连接到牌谱屋数据服务器

所有权限均用于正常功能，不会收集或上传任何个人信息。

## 安全性说明

- ✅ 脚本完全开源，代码可审查
- ✅ 不修改游戏数据，只读取公开信息
- ✅ 不与雀魂服务器直接交互
- ✅ 不收集或上传用户数据
- ✅ 不触碰雀魂反作弊机制
