# 数据探索功能使用指南

## 问题：HTML 工具无法使用

你遇到的 "Failed to fetch" 错误是因为浏览器的 CORS（跨域）限制。本地 HTML 文件无法直接访问外部 API。

## 解决方案：使用游戏内探索模式

我已经在用户脚本中添加了数据探索功能，可以直接在游戏中使用。

---

## 详细步骤

### 第一步：确认脚本已更新

1. 打开 Tampermonkey 管理面板
2. 找到 "雀魂金玉四麻风格分析助手" 脚本
3. 查看版本号是否为 v2.0.0
4. 如果不是，删除旧版本，重新安装：
   - 文件位置：`dist/雀魂金玉四麻风格分析助手-v2.0.0.user.js`
   - 拖拽到浏览器，或在 Tampermonkey 中导入

### 第二步：进入雀魂对局

⚠️ **重要**：必须在对局中，不能在大厅！

1. 打开雀魂游戏（网页版或客户端都可以）
2. 进入一局四人麻将（金玉之间或其他模式都可以）
3. 等待对局开始，看到其他玩家信息

### 第三步：触发探索模式

1. **按下快捷键**：`Ctrl + Shift + D`
   - 三个键同时按下
   - Ctrl（左下角）+ Shift（左下角）+ D 键

2. **打开浏览器控制台**：
   - 按 `F12` 键
   - 或者右键点击页面 -> "检查" -> 切换到 "Console" 标签

1. **复制游戏对象结构**

2. **复制 API 数据**
   - 找到 "【player_extended_stats 完整响应】" 部分
   - 复制完整的 JSON 数据（大括号 `{}` 里的所有内容）
   - 粘贴到 `docs/API_DATA_STRUCTURE.md` 文件中

3. **复制字段清单**

   - 记录所有字段名和类型

## 常见问题

### Q1: 按了 Ctrl+Alt+E 没反应？

**检查清单**：
- ✅ 确认脚本已安装并启用（Tampermonkey 图标显示 "1"）
- ✅ 确认在对局中，不是在大厅
- ✅ 确认按对了键（三个键同时按）
- ✅ 打开了浏览器控制台（F12）
- ✅ 刷新页面重新进入对局试试

### Q2: 控制台没有输出？

**可能原因**：
1. 脚本没有正确安装
2. 不在对局中
3. 控制台被清空了（点击控制台的"清空"按钮旁边的设置，勾选"保留日志"）

### Q3: 显示错误信息？

### Q4: 我不会用控制台？

**简单教程**：
1. 按 F12 键
2. 点击顶部的 "Console" 标签
3. 就能看到输出了
4. 右键点击输出内容可以复制

## 为什么不能用 HTML 工具？

**技术原因**：
- 浏览器的安全机制（CORS）阻止本地文件访问外部 API
- 用户脚本使用 Tampermonkey 的特权 API，可以绕过这个限制

**解决方案**：
- ✅ 使用游戏内探索模式（推荐）
- ⚠️ 启动本地 Web 服务器（复杂）
- ❌ 禁用浏览器 CORS（不安全）
