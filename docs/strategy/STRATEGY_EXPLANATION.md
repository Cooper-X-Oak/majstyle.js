## 关键指标的战术意义

### 1. 净打点效率

**定义**: 每局平均得点 - 每局平均失点，综合反映玩家的实力

**计算公式**: 净打点效率 = 打点效率 - 铳点损失

**典型范围**:

- 金之间: -500 ~ +1000
- 玉之间: 0 ~ +1500

**战术意义**:

- **高净打点效率（>500）**: 玩家综合实力强，每局平均得点远超失点，在进攻和防守上都有优势
- **中等净打点效率（0-500）**: 玩家实力中等，得失基本平衡
- **低净打点效率（<0）**: 玩家综合实力弱，每局平均失点大于得点，存在明显弱点

**对局影响**:

- 净打点效率是最直接的实力指标，直接反映玩家的期望收益
- 对抗高净打点效率的玩家时，正面硬刚期望值不利，建议采取保守策略
- 对抗低净打点效率的玩家时，可以通过积极施压获取优势

**数据来源**: API 字段 `打点效率` - `铳点损失`

---

### 2. 立直率

**定义**: 玩家选择立直的频率

**计算公式**: 立直率 = 立直次数 / 总局数

**典型范围**:

- 金之间: 18%-20%
- 玉之间: 17%-19%

**战术意义**:

- **高立直率（>25%）**: 进攻积极，追求宝牌价值和立直一发，但会暴露听牌信息
- **中等立直率（18%-20%）**: 正常水平，立直判断较为合理
- **低立直率（<17%）**: 保守打法，可能更依赖副露或默听

**对局影响**:

- 立直率本身不决定强弱，需结合立直收支判断质量
- 高立直率 + 高立直收支 = 立直专家，威胁极大
- 高立直率 + 低立直收支 = 过度立直，可以利用

**数据来源**: API 字段 `立直率`

---

### 3. 立直收支

**定义**: 立直后的平均得失点数（和牌得点 - 放铳失点）

**计算公式**: 立直收支 = 立直后平均和牌得点 - 立直后平均放铳失点

**典型范围**:

- 优秀水平: 2000+ 点
- 正常水平: 0-2000 点
- 不佳水平: <0 点

**战术意义**:

- **高立直收支（>2000）**: 立直质量高，立直后和牌率高、打点大、放铳少
- **中等立直收支（0-2000）**: 立直质量正常，立直判断基本合理
- **低立直收支（<0）**: 立直质量差，可能存在过度立直或立直时机不佳的问题

**对局影响**:

- 直接反映立直的期望值，是判断立直威胁的核心指标
- 对手立直收支 >2000 时，立直后建议立即弃牌，放铳期望损失约 7000-8000 点
- 对手立直收支 <0 时，可以适度对抗，不必过度防守

**推理基础**:

- 立直收支 = 立直后和牌率 × 平均打点 - 立直后放铳率 × 平均铳点
- 高立直收支意味着：立直后和牌率高（容易和牌）+ 立直后放铳率低（不容易放铳）+ 平均打点可能较高（和牌价值大）

**数据来源**: API 字段 `立直收支`

---

### 4. 副露率

**定义**: 玩家选择副露（吃、碰、杠）的频率

**计算公式**: 副露率 = 副露次数 / 总局数

**典型范围**:

- 金之间: 33%-36%
- 玉之间: 32%-34%

**战术意义**:

- **高副露率（>38%）**: 积极副露，追求速度和确定性，但会降低打点和灵活性
- **中等副露率（32%-36%）**: 正常水平，副露判断较为合理
- **低副露率（<30%）**: 保守副露，可能更依赖立直或默听

**对局影响**:

- 副露率本身不决定强弱，需结合副露后和牌率判断质量
- 高副露率 + 高副露后和牌率 = 副露专家，速攻压力大
- 高副露率 + 低副露后和牌率 = 副露判断不佳，可以施压

**数据来源**: API 字段 `副露率`

---

### 5. 副露后和牌率

**定义**: 副露后成功和牌的概率

**计算公式**: 副露后和牌率 = 副露后和牌次数 / 副露次数

**典型范围**:

- 优秀水平: 35%+
- 正常水平: 28%-35%
- 不佳水平: <25%

**战术意义**:

- **高副露后和牌率（>35%）**: 副露判断优秀，副露后能高效完成和牌
- **中等副露后和牌率（28%-35%）**: 副露质量正常
- **低副露后和牌率（<25%）**: 副露判断不佳，副露后容易陷入困境

**对局影响**:

- 对手副露后和牌率 >35% 时，副露后威胁大，需要警惕对手通过副露抢先和牌
- 对手副露后和牌率 <25% 时，副露后可以积极施压，迫使对手陷入不利局面

**数据来源**: API 字段 `副露后和牌率`

---

### 6. 默听率

**定义**: 不立直、不副露而直接和牌的频率

**计算公式**: 默听率 ≈ 和牌率 - 立直率 - 副露率（简化估算）

**典型范围**:

- 高隐蔽: 15%+
- 中等: 8%-15%
- 低隐蔽: <8%

**战术意义**:

- **高默听率（>15%）**: 擅长隐蔽听牌，经常在无明显征兆时突然和牌
- **中等默听率（8%-15%）**: 正常水平
- **低默听率（<8%）**: 进攻意图明显，多通过立直或副露和牌

**对局影响**:

- 对手默听率 >12% 时，需要全程保持警惕，不能因为对手未立直就放松防守
- 高默听率玩家的进攻往往无征兆，难以预判

**推理基础**:

- 默听 = 门清状态下不立直而和牌
- 高默听率说明玩家擅长判断何时不立直更有利（避免暴露、追求默听自摸、顺位保护等）

**数据来源**: API 字段 `默听率`（或通过 `和牌率 - 立直率 - 副露率` 估算）

---

### 7. 和了巡数

**定义**: 玩家和牌时的平均巡目数

**典型范围**:

- 快速: <11 巡
- 中速: 11-13 巡
- 慢速: >13 巡

**战术意义**:

- **快速（<11 巡）**: 速攻型打法，能快速完成听牌和和牌，抢先进攻压力大
- **中速（11-13 巡）**: 正常速度
- **慢速（>13 巡）**: 倾向于慢速布局或追求高打点，给对手留出充足时间

**对局影响**:

- 对手和了巡数 <11 时，需要在前期就做好防守准备，或者通过更快的速度抢先和牌
- 对手和了巡数 >13 时，有充足的布局时间，可以追求更高的牌型价值

**数据来源**: API 字段 `和了巡数`

---

### 8. 放铳率

**定义**: 玩家放铳的频率

**计算公式**: 放铳率 = 放铳次数 / 总局数

**典型范围**:

- 优秀水平: <13%
- 正常水平: 13%-15%
- 不佳水平: >16%

**战术意义**:

- **低放铳率（<13%）**: 防守能力强，能有效避免放铳
- **中等放铳率（13%-15%）**: 正常水平
- **高放铳率（>16%）**: 防守能力弱，容易放铳

**对局影响**:

- 对手放铳率 <13% 时，难以通过普通牌型让对手放铳，需要更高的牌型质量
- 对手放铳率 >16% 时，可以通过积极进攻施压，增加对手放铳概率

**数据来源**: API 字段 `放铳率`

---

### 9. 平均铳点

**定义**: 玩家放铳时的平均点数

**典型范围**:

- 优秀水平: <5000 点
- 正常水平: 5000-5500 点
- 不佳水平: >5500 点

**战术意义**:

- **低平均铳点（<5000）**: 即使放铳也多为小牌，大牌防守能力强
- **高平均铳点（>5500）**: 容易放大牌，大牌防守能力差

**对局影响**:

- 对手平均铳点 <5000 时，难以通过大牌获利
- 对手平均铳点 >5500 时，可以追求高打点牌型

**数据来源**: API 字段 `平均铳点`

---

### 10. 被炸率

**定义**: 放铳中满贯以上的比例

**计算公式**: 被炸率 = 满贯以上放铳次数 / 总放铳次数

**典型范围**:

- 优秀水平: <8%
- 正常水平: 8%-10%
- 不佳水平: >10%

**战术意义**:

- **低被炸率（<8%）**: 大牌防守能力强，能有效识别和避免大牌
- **高被炸率（>10%）**: 大牌防守能力差，容易放大牌

**对局影响**:

- 对手被炸率 >10% 时，可以追求高打点牌型（满贯、跳满、倍满），对手放铳概率较高

**数据来源**: API 字段 `被炸率`

## 玩家原型详解

本系统定义了 6 种核心玩家原型，每种原型都有明确的匹配条件和专属策略建议。

### 原型 1: 立直专家 🎯

**定义条件**:

- 立直率 ≥ 25%（显著高于平均 19%）
- 立直收支 ≥ 1500（立直质量优秀）
- 副露率 ≤ 30%（不依赖副露）

**战术特征**:

- 擅长判断立直时机，立直质量高
- 立直后和牌率高、放铳率低
- 倾向追求宝牌价值和立直一发
- 不擅长副露速攻

**典型数据**:

- 立直后和牌率: 45%-50%（普通玩家 40%-42%）
- 立直后放铳率: 7%-9%（普通玩家 11%-13%）
- 立直平均打点: 7000-8000（普通玩家 6500）

**对局策略**:

1. **立直后立即弃牌**: 对手立直质量高，放铳风险大，期望损失约 7000-8000 点
2. **警惕好型立直**: 对手可能有高打点听牌，避免追求和牌
3. **可通过副露抢先**: 对手不擅长副露，副露可抢先和牌，打乱对手节奏

**理论依据**:

- 立直收支 = 立直后和牌率 × 平均打点 - 立直后放铳率 × 平均铳点
- 高立直收支意味着立直期望值高，对抗时防守期望值更优

**阈值依据**:

- 立直率 25%: 约为平均值（19%）的 1.3 倍，显著偏高
- 立直收支 1500: 约为正常水平（0-1000）的 1.5 倍以上
- 副露率 30%: 低于平均值（35%），说明不依赖副露

---

### 原型 2: 副露专家 ⚡

**定义条件**:

- 副露率 ≥ 38%（显著高于平均 35%）
- 副露后和牌率 ≥ 30%（副露质量优秀）
- 立直率 ≤ 20%（不依赖立直）

**战术特征**:

- 擅长判断副露时机，副露质量高
- 副露后能高效完成和牌
- 追求速度和确定性
- 不擅长立直进攻

**典型数据**:

- 副露后和牌率: 32%-38%（普通玩家 28%-30%）
- 副露后放铳率: 12%-15%（普通玩家 15%-18%）
- 和了巡数: 10-11 巡（普通玩家 12 巡）

**对局策略**:

1. **副露后警惕速攻**: 对手副露效率高，副露后威胁大，需要快速防守
2. **可通过立直施压**: 对手不擅长立直，立直可以施压并获取主动权
3. **注意抢先进攻**: 对手速度快，需要在前期就做好准备

**理论依据**:

- 副露后和牌率直接反映副露判断质量
- 高副露后和牌率意味着副露期望值高，速攻压力大

**阈值依据**:

- 副露率 38%: 约为平均值（35%）的 1.1 倍
- 副露后和牌率 30%: 高于平均值（28%），说明副露质量优秀
- 立直率 20%: 低于平均值（19%），说明不依赖立直

---

### 原型 3: 默听猎手 🥷

**定义条件**:

- 默听率 ≥ 15%（显著高于平均 8%-10%）
- 立直率 ≤ 20%（不依赖立直）
- 放铳率 ≤ 14%（防守能力强）

**战术特征**:

- 擅长隐蔽听牌，经常在无明显征兆时突然和牌
- 不暴露听牌信息，保持灵活性
- 防守能力强，不容易放铳
- 难以预判进攻时机

**典型数据**:

- 默听率: 15%-20%（普通玩家 8%-10%）
- 立直率: 15%-18%（普通玩家 19%）
- 放铳率: 12%-14%（普通玩家 15%）

**对局策略**:

1. **全程保持警惕**: 对手进攻无征兆，不能因为对手未立直就放松防守
2. **避免过度进攻**: 对手防守能力强，过度进攻容易陷入不利
3. **注意顺位变化**: 对手可能通过默听保护顺位或追求高打点

**理论依据**:

- 默听 = 门清状态下不立直而和牌
- 高默听率说明玩家擅长判断何时不立直更有利（避免暴露、追求默听自摸、顺位保护等）

**阈值依据**:

- 默听率 15%: 约为平均值（8%-10%）的 1.5-2 倍
- 立直率 20%: 低于平均值（19%），说明不依赖立直
- 放铳率 14%: 低于平均值（15%），说明防守能力强

---

### 原型 4: 速攻型 💨

**定义条件**:

- 和了巡数 ≤ 11（显著快于平均 12 巡）
- 副露率 ≥ 35%（积极副露）
- 和牌率 ≥ 21%（和牌频率高）

**战术特征**:

- 追求速度，能快速完成听牌和和牌
- 积极副露，抢先进攻
- 和牌频率高，但打点可能较小
- 前期压力大

**典型数据**:

- 和了巡数: 9-11 巡（普通玩家 12 巡）
- 副露率: 35%-40%（普通玩家 35%）
- 和牌率: 22%-25%（普通玩家 21%）

**对局策略**:

1. **前期做好防守**: 对手速度快，需要在前期就做好防守准备
2. **可追求高打点**: 对手追求速度，打点可能较小，可以通过高打点反制
3. **注意抢先进攻**: 如果能比对手更快，可以抢先和牌

**理论依据**:

- 和了巡数直接反映速度
- 速攻型玩家通过高频率和牌获取优势，但单次得点可能较小

**阈值依据**:

- 和了巡数 11: 低于平均值（12 巡），说明速度快
- 副露率 35%: 平均水平或以上，说明积极副露
- 和牌率 21%: 平均水平或以上，说明和牌频率高

---

### 原型 5: 价值型 💎

**定义条件**:

- 平均打点 ≥ 7000（显著高于平均 6500）
- 立直率 ≥ 20%（积极立直）
- 和了巡数 ≥ 12（不追求速度）

**战术特征**:

- 追求高打点，不追求速度
- 倾向于慢速布局，追求宝牌和役种
- 立直频率高，追求立直一发和宝牌价值
- 单次和牌得点高

**典型数据**:

- 平均打点: 7000-8000（普通玩家 6500）
- 立直率: 20%-25%（普通玩家 19%）
- 和了巡数: 12-14 巡（普通玩家 12 巡）

**对局策略**:

1. **警惕大牌**: 对手追求高打点，放铳损失可能很大
2. **可通过速攻抢先**: 对手速度慢，可以通过速攻抢先和牌
3. **注意宝牌**: 对手可能追求宝牌，警惕宝牌相关的危险牌

**理论依据**:

- 平均打点直接反映打点追求
- 价值型玩家通过高打点获取优势，但和牌频率可能较低

**阈值依据**:

- 平均打点 7000: 高于平均值（6500），说明追求高打点
- 立直率 20%: 平均水平或以上，说明积极立直
- 和了巡数 12: 平均水平或以上，说明不追求速度

---

### 原型 6: 防守型 🛡️

**定义条件**:

- 放铳率 ≤ 13%（显著低于平均 15%）
- 被炸率 ≤ 8%（大牌防守能力强）
- 默听率 ≥ 12%（倾向默听）

**战术特征**:

- 防守能力强，能有效避免放铳
- 大牌防守能力强，不容易放大牌
- 倾向默听，保持灵活性
- 进攻可能较为保守

**典型数据**:

- 放铳率: 11%-13%（普通玩家 15%）
- 被炸率: 6%-8%（普通玩家 9%-10%）
- 默听率: 12%-15%（普通玩家 8%-10%）

**对局策略**:

1. **需要更高牌型质量**: 对手防守能力强，难以通过普通牌型让对手放铳
2. **可积极进攻**: 对手进攻可能较为保守，可以积极进攻获取主动权
3. **注意默听**: 对手倾向默听，需要全程保持警惕

**理论依据**:

- 放铳率和被炸率直接反映防守能力
- 防守型玩家通过减少失点获取优势

**阈值依据**:

- 放铳率 13%: 低于平均值（15%），说明防守能力强
- 被炸率 8%: 低于平均值（9%-10%），说明大牌防守能力强
- 默听率 12%: 高于平均值（8%-10%），说明倾向默听

## 策略建议的生成逻辑

本节详细解释每条策略建议的触发条件、推理过程、数据支撑和战术建议。

### 建议 1: "对手立直质量高，立直后建议立即弃牌"

**触发条件**:

- 立直收支 > 2000 点
- 或：立直后和牌率 > 45% 且 立直后放铳率 < 10%

**推理过程**:

1. 立直收支 = 立直后和牌得点 - 立直后放铳失点
2. 高立直收支意味着：
   - 立直后和牌率高（容易和牌）
   - 立直后放铳率低（不容易放铳）
   - 平均打点可能较高（和牌价值大）
3. 综合结果：对手立直后威胁大，放铳期望损失高

**数据支撑**:

- 立直收支 > 2000 的玩家，立直后和牌率平均 48%（普通玩家 42%）
- 立直后放铳率平均 8%（普通玩家 12%）
- 对手立直后，放铳期望损失约 7000-8000 点
- 弃牌可避免约 15% 的放铳概率

**战术建议**:

- 对手立直后，立即切换到防守模式
- 优先打安全牌（现物、筋牌）
- 避免追求和牌，除非已经听牌且牌型优秀
- 如果手牌很差，考虑直接弃牌保护点数

**期望值计算**:

- 继续进攻期望值: 和牌概率 × 和牌得点 - 放铳概率 × 放铳失点 ≈ 10% × 6000 - 15% × 7500 = -525 点
- 弃牌期望值: 0 点（保护点数）
- 结论：弃牌期望值更优

---

### 建议 2: "对手综合实力强，不建议正面硬刚"

**触发条件**:

- 净打点效率 > 500 点

**推理过程**:

1. 净打点效率 = 打点效率 - 铳点损失
2. 高净打点效率意味着：
   - 每局平均得点远超失点
   - 在进攻和防守上都有优势
3. 综合结果：对手综合实力强，正面对抗期望值不利

**数据支撑**:

- 净打点效率 > 500 的玩家，平均顺位约 2.3（普通玩家 2.5）
- 和牌率通常较高（22%-25%）
- 放铳率通常较低（13%-14%）

**战术建议**:

- 采取保守策略，避免正面硬刚
- 优先保护点数，减少失点
- 等待对手失误或牌运不佳的机会
- 如果顺位落后，可以适度冒险，但需要谨慎

**期望值计算**:

- 正面对抗期望值: 对手实力强，期望值不利
- 保守策略期望值: 减少失点，等待机会

### 建议 3: "对手有较高默听倾向，警惕无征兆进攻"

**触发条件**:

- 默听率 > 12%

**推理过程**:

1. 默听率 = 和牌率 - 立直率 - 副露率（简化估算）
2. 高默听率意味着：
   - 对手经常不立直、不副露而直接和牌
   - 进攻无明显征兆
3. 综合结果：对手进攻难以预判，需要全程警惕

**数据支撑**:

- 默听率 > 12% 的玩家，默听和牌占总和牌的 40%-50%
- 默听和牌往往无征兆，难以预判

**战术建议**:

- 全程保持警惕，不能因为对手未立直就放松防守
- 注意对手的舍牌变化，警惕突然的防守转换
- 如果对手长时间未鸣牌，可能已经听牌
- 避免打危险牌，即使对手未立直

### 建议 4: "对手速度快，需要抢先进攻"

**触发条件**:

- 和了巡数 < 11 巡

**推理过程**:

1. 和了巡数直接反映速度
2. 低和了巡数意味着：
   - 对手能快速完成听牌和和牌
   - 前期压力大
3. 综合结果：需要在前期就做好准备，或者通过更快的速度抢先和牌

**数据支撑**:

- 和了巡数 < 11 的玩家，通常在 9-10 巡就能和牌
- 副露率通常较高（35%-40%）
- 和牌率通常较高（22%-25%）

**战术建议**:

- 前期做好防守准备，警惕对手的快速进攻
- 如果能比对手更快，可以抢先和牌，打乱对手节奏
- 避免慢速布局，否则容易被对手抢先
- 如果手牌很差，考虑早期弃牌

---

### 建议 5: "对手副露效率高，副露后威胁大"

**触发条件**:

- 副露后和牌率 > 35%

**推理过程**:

1. 副露后和牌率直接反映副露判断质量
2. 高副露后和牌率意味着：
   - 对手擅长判断副露时机
   - 副露后能高效完成和牌
3. 综合结果：对手副露后威胁大，需要快速防守

**数据支撑**:

- 副露后和牌率 > 35% 的玩家，副露后和牌概率约 35%-40%（普通玩家 28%-30%）
- 副露后放铳率通常较低（12%-15%）

**战术建议**:

- 对手副露后，立即切换到防守模式
- 警惕对手的快速进攻，避免放铳
- 可以通过立直施压，打乱对手节奏
- 如果手牌很差，考虑直接弃牌

### 建议 6: "对手防守能力强，需要更高牌型质量"

**触发条件**:

- 放铳率 < 13% 且 平均铳点 < 5000

**推理过程**:

1. 放铳率和平均铳点直接反映防守能力
2. 低放铳率和低平均铳点意味着：
   - 对手能有效避免放铳
   - 即使放铳也多为小牌
3. 综合结果：对手防守能力强，难以通过普通牌型让对手放铳

**数据支撑**:

- 放铳率 < 13% 的玩家，放铳概率约 11%-13%（普通玩家 15%）
- 平均铳点 < 5000 的玩家，放铳多为小牌（3900-4000）

**战术建议**:

- 需要更高的牌型质量才能让对手放铳
- 追求高打点牌型（满贯、跳满、倍满）
- 避免追求小牌，期望值不高
- 可以通过立直施压，迫使对手防守

---

### 建议 7: "对手被炸率高，大牌防守差，可以做大牌"

**触发条件**:

- 被炸率 > 10%

**推理过程**:

1. 被炸率 = 满贯以上放铳次数 / 总放铳次数
2. 高被炸率意味着：
   - 对手对大牌的防守能力差
   - 容易放大牌
3. 综合结果：可以追求高打点牌型，对手放铳概率较高

**数据支撑**:

- 被炸率 > 10% 的玩家，放铳中约 10%-15% 为满贯以上
- 平均铳点通常较高（5500-6000）

**战术建议**:

- 追求高打点牌型（满贯、跳满、倍满）
- 不必过度追求速度，可以慢速布局
- 警惕对手的大牌，但对手放大牌的概率也高
- 如果已经听牌且打点高，可以积极施压

## 危险度评分系统

危险度评分系统是一个 10 维度的综合评估系统，用于量化对手的威胁程度。

### 评分维度和权重

| 维度     | 权重  | 说明         |
| ------ | --- | ---------- |
| 净打点效率  | 30% | 综合实力指标，最重要 |
| 立直收支   | 20% | 立直质量，威胁大   |
| 默听率    | 15% | 隐蔽性，难以预判   |
| 和了巡数   | 10% | 速度，前期压力    |
| 被炸率    | 10% | 大牌防守能力（反向） |
| 立直后和牌率 | 5%  | 立直效率       |
| 平均铳点   | 5%  | 防守弱点（反向）   |
| 副露后和牌率 | 3%  | 副露效率       |
| 和牌率    | 2%  | 基础进攻能力     |

**总权重**: 100%

### 评分计算方法

每个维度的得分范围为 0-10 分，最终危险度 = 各维度得分 × 权重之和 × 置信度调整

**归一化公式**:

```
normalized_score = (value - min) / (max - min)
score = normalized_score × 10
```

**示例**:

- 净打点效率: 范围 -1000 ~ 2000，值 1000 → 归一化 (1000 - (-1000)) / (2000 - (-1000)) = 0.67 → 得分 6.7
- 默听率: 范围 0 ~ 0.2，值 0.15 → 得分 0.15 × 50 = 7.5

### 置信度调整

根据样本量调整最终得分，避免小样本偏差：

| 样本量       | 置信度  |
| --------- | ---- |
| ≥ 400 局   | 100% |
| 200-399 局 | 95%  |
| 100-199 局 | 90%  |
| 50-99 局   | 80%  |
| < 50 局    | 70%  |

**最终危险度** = 加权得分 × 置信度

### 危险度等级

| 分数   | 等级   | 图标  | 说明        |
| ---- | ---- | --- | --------- |
| 9-10 | 极度危险 | ⚠️  | 顶级高手，避免对抗 |
| 7-8  | 较为危险 | ⚡   | 实力强劲，谨慎应对 |
| 5-6  | 正常水平 | ➡️  | 普通对手，正常对局 |
| 3-4  | 威胁较小 | ✓   | 实力较弱，可以施压 |
| 0-2  | 送分目标 | 🎯  | 明显弱点，积极进攻 |

### 权重设定依据

**当前状态**: ⚠️ 待验证（基于经验值，需要后续数据验证）

**初步理由**:

1. **净打点效率 30%**: 综合实力指标，直接反映期望收益，最重要
2. **立直收支 20%**: 立直是最常见的进攻方式，立直质量直接影响威胁程度
3. **默听率 15%**: 隐蔽性高的对手难以预判，威胁大
4. **和了巡数 10%**: 速度快的对手前期压力大
5. **被炸率 10%**: 大牌防守能力差的对手容易放大牌，但这是反向指标
6. **其他维度 5% 以下**: 辅助指标，影响相对较小

**改进方向**:

- 通过回归分析确定最优权重
- 验证各维度与胜率的相关性
- 根据实际数据调整权重

## 附录: 数据来源和计算方法

### API 数据来源

所有数据来自牌谱屋（Amae Koromo）公开 API:

- 基础 URL: `https://5-data.amae-koromo.com/api/v2/pl4`
- 端点: `/player_stats/{id}/{start}/{end}`, `/player_extended_stats/{id}/{start}/{end}`
- 模式: `12.9` (四人金玉段位)
- 标签: `492541` (金之间/玉之间)

### 计算方法

**净打点效率**:

```
净打点效率 = 打点效率 - 铳点损失
打点效率 = (和牌总得点 + 自摸总得点) / 总局数
铳点损失 = 放铳总失点 / 总局数
```

**默听率**:

```
默听率 ≈ 和牌率 - 立直率 - 副露率（简化估算）
或: 默听率 = 默听和牌次数 / 总局数（精确计算）
```

**立直收支**:

```
立直收支 = 立直后平均和牌得点 - 立直后平均放铳失点
```

**被炸率**:

```
被炸率 = 满贯以上放铳次数 / 总放铳次数
```
