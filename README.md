# 雀魂金玉四麻风格分析助手

实时分析雀魂金之间/玉之间对手风格的 Tampermonkey 脚本，提供 10 维度危险度评估、6 种玩家原型检测和实时战术建议。

## 项目简介

这是一个专为雀魂（Mahjong Soul）金之间和玉之间四人麻将对局设计的用户脚本，通过分析对手的历史数据，实时显示其打法风格和战术建议。

**核心特点**:
- 基于牌谱屋公开 API，完全合规，不涉及任何破解或修改游戏行为
- 仅读取游戏公开暴露的数据（`window.view.DesktopMgr.Inst.player_datas`）
- 半透明悬浮界面，不遮挡游戏画面

## 功能特性

### 实时风格分析
- ✅ **9 种主称号分类**：钢铁战士、狂战士、自爆兵、忍者、上班族、赌徒、乌龟、摆烂人、送分童子
- ✅ **11 种特征标签**：精准狙击、速攻流、立直狂、大牌猎人、对攻狂魔等
- ✅ **颜色编码偏差值**：红色（高于平均）、绿色（低于平均），深浅表示偏差程度

### 玩家原型检测（Phase 1）
- 🎯 **6 种核心原型**：立直专家、副露专家、默听猎手、速攻型、价值型、防守型
- 🔍 **智能匹配算法**：基于多维度条件自动识别玩家打法风格
- 📊 **匹配度评分**：显示原型匹配置信度
- 🎨 **原型图标**：每种原型配有专属 emoji 图标

### 危险度评估系统（Phase 1）
- ⚡ **10 维度评估**：净打点效率、默听率、立直收支、平均铳点、被炸率、和了巡数等
- 📈 **置信度系统**：根据对局数量（50-400+ 局）动态调整评分可靠性
- 🎯 **归一化评分**：所有维度统一归一化到 0-10 分，确保公平性

### 战术建议生成
- 🧠 **上下文感知建议**：根据检测到的玩家原型生成针对性建议
- 📋 **原型专属建议库**：每种原型配有 3 条专属战术建议
- 🔄 **智能去重**：避免重复建议，优先显示原型特定建议

### 用户界面
- 🎨 **半透明悬浮窗**：60% 透明度，不遮挡游戏画面
- 📍 **智能位置分配**：自己固定左下角，其他玩家动态分配
- 🔄 **自动清理**：退出房间自动清除信息框

## 快速开始

### 1. 安装 Tampermonkey

在浏览器中安装 Tampermonkey 扩展：
- [Chrome/Edge](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox](https://addons.mozilla.org/firefox/addon/tampermonkey/)

### 2. 安装脚本

1. 下载最新版本的脚本文件：`dist/雀魂金玉四麻风格分析助手-v2.1.0-beta.1.user.js`
2. 将文件拖拽到浏览器窗口，或在 Tampermonkey 管理面板中导入
3. 点击"安装"按钮

### 3. 开始使用

1. 打开雀魂游戏（网页版或客户端都可以）
2. 进入金之间或玉之间的四人麻将对局
3. 等待 2-3 秒，脚本自动分析并显示对手风格信息

详细安装步骤请查看 [安装指南](docs/user/installation.md)
