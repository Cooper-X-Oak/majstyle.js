# 雀魂金玉四麻风格分析助手

一个基于 Tampermonkey 的雀魂对手风格实时分析工具，专注于金之间和玉之间的四人麻将对局。

## ✨ 功能特点

- 🎯 **实时风格分析**：自动识别对局玩家，分析其历史数据
- 📊 **数据可视化**：半透明悬浮窗显示关键数据，不遮挡游戏画面
- 🏷️ **风格标签系统**：9种主称号 + 11种特征标签，快速了解对手
- 📈 **偏差对比**：所有数据对比段位平均值，颜色标注偏差程度
- ⚡ **快速响应**：进入对局后2-3秒内完成分析

## 🎮 支持范围

- **对局类型**：仅支持四人麻将（四麻）
- **段位范围**：金之间、玉之间（士3-魂1）
- **数据来源**：牌谱屋（Amae Koromo）公开 API

## 📦 安装方法

### 快速安装（推荐）

**一键安装链接**：[点击安装脚本](https://github.com/Cooper-X-Oak/majstyle.js/raw/main/雀魂金玉四麻风格分析助手-v2.0.0.user.js)

> 前提：需要先安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展

点击上方链接后，Tampermonkey 会自动识别并弹出安装确认页面，点击"安装"即可。

### 手动安装

#### 1. 安装 Tampermonkey

- Chrome/Edge: [Tampermonkey 官网](https://www.tampermonkey.net/)
- Firefox: [Firefox 插件商店](https://addons.mozilla.org/firefox/addon/tampermonkey/)

#### 2. 安装脚本

1. 点击 Tampermonkey 图标 → 管理面板
2. 点击"+"号创建新脚本
3. 复制 `雀魂金玉四麻风格分析助手-v2.0.0.user.js` 的全部内容
4. 粘贴到编辑器，保存（Ctrl+S）

### 使用

1. 打开雀魂游戏（https://game.maj-soul.com 或 https://game.maj-soul.net）
2. 进入金之间或玉之间的四麻对局
3. 等待2-3秒，自动显示分析结果

详细安装指南请参考 [docs/installation.md](docs/installation.md)

## 📊 数据说明

### 显示内容

- **主称号**：根据进攻意愿和防守能力分类
  - 高意愿：钢铁战士（铁壁）、狂战士（正常）、自爆兵（漏勺）
  - 中意愿：忍者（铁壁）、上班族（正常）、赌徒（漏勺）
  - 低意愿：乌龟（铁壁）、摆烂人（正常）、送分童子（漏勺）

- **特征标签**：精准狙击、速攻流、立直狂、大牌猎人、对攻狂魔、慢热型、铁壁、漏勺、先制王、好型立直、赌博立直

- **核心数据**（括号内为偏差值）：立直率、副露率、和牌率、放铳率、平均打点

### 颜色编码

- 🔴 **红色**：高于段位平均（正偏差）
- 🟢 **绿色**：低于段位平均（负偏差）
- 颜色深浅表示偏差程度

详细使用说明请参考 [docs/usage.md](docs/usage.md)

## 🔒 合规性说明

### 为什么 GitHub 上的雀魂项目都挂了？

大部分开源项目试图直接解析雀魂的 WebSocket 通信、破解 protobuf 协议、注入代码修改游戏行为，这些都会触发雀魂的反作弊检测。

### 本项目的合规方案

✅ **完全合规，不触碰任何反爬机制**

1. **获取玩家 ID**：通过浏览器开发者工具观察游戏对象，读取 `window.view.DesktopMgr.Inst.player_datas`，这是游戏本身暴露的公开数据
2. **数据来源**：调用牌谱屋（Amae Koromo）的公开 API
3. **完全绕过雀魂**：不直接与雀魂服务器交互、不解析 WebSocket、不破解 protobuf、不修改游戏数据

## 🐛 常见问题

**Q: 为什么有的玩家显示"无牌谱数据"？**
A: 可能是新号、没打过金之间、或对局数不足 50 局。

**Q: 数据多久更新一次？**
A: 牌谱屋的数据通常有几小时到一天的延迟，不是实时的。

**Q: 会被封号吗？**
A: 不会。脚本完全不触碰雀魂的反作弊机制，只读取公开数据。

**Q: 可以用于三麻吗？**
A: 不可以。所有基准数据都是基于四麻金玉之间，三麻的数据分布完全不同。
