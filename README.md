## 项目简介

这是一个专为雀魂（Mahjong Soul）四人麻将对局设计的用户脚本，通过分析对手的历史数据，实时显示其打法风格和战术建议。

**核心特点**:
- 完全合规，不涉及任何破解或修改游戏行为
- 仅读取游戏公开暴露的数据（`window.view.DesktopMgr.Inst.player_datas`）
- 半透明悬浮界面，可展开/收起，不影响游戏操作
- **支持所有段位**：从铜之间到魂天，只要有足够的对局数据（50+ 局）即可分析

## 功能特性

### Phase 2.2: 进攻/防守强度评估系统 ⭐ NEW
- ⚔️ **进攻强度评估**：基于立直进攻、副露进攻、速度、打点 4 个维度，0-100 分评分，映射到 Lv.1-10 等级
- 🛡️ **防守强度评估**：基于放铳控制、大牌防守、隐蔽性、立直防守 4 个维度，0-100 分评分，映射到 Lv.1-10 等级
- 🎭 **态度词生成**：根据突出特征自动生成态度词（立直狂、副露狂、速攻狂、铁壁等）
- 🎨 **等级颜色编码**：金色（Lv.9-10）、紫色（Lv.7-8）、蓝色（Lv.5-6）、灰色（Lv.1-4）
- 📊 **配置化系统**：所有阈值、权重、公式都在 JSON 配置文件中，无需改代码即可调整

### 实时风格分析
- ✅ **9 种主称号分类**：钢铁战士、狂战士、自爆兵、忍者、上班族、赌徒、乌龟、摆烂人、送分童子
- ✅ **11 种特征标签**：精准狙击、速攻流、立直狂、大牌猎人、对攻狂魔等
- ✅ **颜色编码偏差值**：红色（高于平均）、绿色（低于平均），深浅表示偏差程度

### 玩家原型检测（Phase 1）
- 🎯 **6 种核心原型**：立直专家、副露专家、默听猎手、速攻型、价值型、防守型
- 🔍 **智能匹配算法**：基于多维度条件自动识别玩家打法风格
- 📊 **匹配度评分**：显示原型匹配置信度
- 🎨 **原型图标**：每种原型配有专属 emoji 图标

### 危险度评估系统（Phase 1）
- ⚡ **10 维度评估**：净打点效率、默听率、立直收支、平均铳点、被炸率、和了巡数等
- 📈 **置信度系统**：根据对局数量（50-400+ 局）动态调整评分可靠性
- 🎯 **归一化评分**：所有维度统一归一化到 0-10 分，确保公平性

### 战术建议生成
- 🧠 **上下文感知建议**：根据检测到的玩家原型生成针对性建议
- 📋 **原型专属建议库**：每种原型配有 3 条专属战术建议
- 🔄 **智能去重**：避免重复建议，优先显示原型特定建议

### 用户界面
- 🎨 **可展开/收起面板**：默认精简模式，点击展开查看详细信息
- 📍 **智能位置分配**：自己固定左下角，其他玩家动态分配
- 🔄 **自动清理**：退出房间自动清除信息框
- ⚡ **平滑动画**：展开/收起带有过渡动画，体验流畅

## 快速开始

### 1. 安装 Tampermonkey

在浏览器中安装 Tampermonkey 扩展：
- [Chrome/Edge](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox](https://addons.mozilla.org/firefox/addon/tampermonkey/)

### 2. 安装脚本

**方式一：一键安装（推荐）**

点击下方链接，Tampermonkey 会自动打开安装页面：

👉 [**点击安装最新版本**](https://github.com/Cooper-X-Oak/majstyle.js/raw/main/dist/雀魂四麻风格分析助手-v2.2.0-beta.2.user.js)

**方式二：手动安装**

1. 下载最新版本的脚本文件：`dist/雀魂四麻风格分析助手-v2.2.0-beta.2.user.js`
2. 将文件拖拽到浏览器窗口，或在 Tampermonkey 管理面板中导入
3. 点击"安装"按钮

### 3. 开始使用

1. 打开雀魂游戏（网页版或客户端都可以）
2. 进入任意段位的四人麻将对局（铜之间到魂天均可）
3. 等待 2-3 秒，脚本自动分析并显示对手风格信息
4. **点击面板可展开/收起详细信息**

**注意**：
- 需要对手有 50+ 局对局数据才能进行分析
- 段位基准数据目前覆盖士3-魂1，其他段位使用杰3作为默认基准

详细安装步骤请查看 [安装指南](docs/user/installation.md)

## 反馈与建议

### 🎯 我们需要你的帮助！

这个项目的分析算法还在持续优化中，**我们非常需要实战玩家的反馈**来改进算法的准确性和实用性。

#### 请帮助我们回答以下问题：

1. **强度评估准确性**：
   - 进攻/防守强度等级（Lv.1-10）是否符合你的实际观察？
   - 态度词（立直狂、副露狂、铁壁等）是否准确描述了对手特征？
   - 有没有遇到"明明很强但评级很低"或"明明很弱但评级很高"的情况？

2. **危险度评估有效性**：
   - 危险度评分（0-10 分）是否帮助你做出正确的战术决策？
   - 哪些维度的权重需要调整？（当前：净打点效率 30%、默听率 15%、立直收支 20%...）

3. **玩家原型识别**：
   - 6 种原型（立直专家、副露专家、默听猎手、速攻型、价值型、防守型）是否覆盖了你遇到的大部分对手？
   - 有没有遗漏的重要打法风格？

4. **策略建议实用性**：
   - 策略建议是否具有可操作性？
   - 有没有遇到"建议不适用"或"建议错误"的情况？

5. **数据利用率**：
   - 当前使用了 26/51 个 API 字段（51% 利用率），还有 25 个字段未使用（如一发率、里宝率、流局率等）
   - 你认为哪些未使用的字段对分析最有价值？
