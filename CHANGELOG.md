# CHANGELOG

## [2.2.0-beta.2] - 2026-03-15 (测试版 - Phase 2.2 强度评估系统)

### Phase 2.2: 进攻/防守强度评估系统

⚠️ **这是测试版本，实现基于数据推理的强度评估，用"态度词 + 等级"展示**

#### 新增功能

**强度评估系统**
- ⚔️ **进攻强度评估** (0-100 分，Lv.1-10)
  - 立直进攻 (30% 权重): 基于立直率、立直收支、立直后和牌率
  - 副露进攻 (25% 权重): 基于副露率、副露后和牌率
  - 速度 (25% 权重): 基于和了巡数
  - 打点 (20% 权重): 基于平均打点、自摸率
- 🛡️ **防守强度评估** (0-100 分，Lv.1-10)
  - 放铳控制 (40% 权重): 基于放铳率、平均铳点
  - 大牌防守 (30% 权重): 基于被炸率
  - 隐蔽性 (20% 权重): 基于默听率
  - 立直防守 (10% 权重): 基于立直后放铳率

**态度词生成**
- 进攻态度词: 立直狂、副露狂、速攻狂、大牌型等
- 防守态度词: 铁壁、防守稳健、隐蔽高手、防守弱等
- 支持组合态度词（如"立直狂 + 速攻狂"）

**等级映射**
- Lv.1-2: 初学者水平（灰色）
- Lv.3-4: 青铜水平（灰色）
- Lv.5-6: 白银水平（蓝色）
- Lv.7-8: 黄金水平（紫色）
- Lv.9: 王者水平（金色）
- Lv.10: 传说水平（金色）

**完全配置化**
- ✅ 所有权重配置化（进攻/防守子维度权重）
- ✅ 所有归一化范围配置化（立直率 10%-30%、和了巡数 14-9 巡等）
- ✅ 所有态度词阈值配置化（立直狂、铁壁等触发条件）
- ✅ 无任何硬编码，全部从 `analysis-config.json` 读取

**UI 展示增强**
- 在玩家卡片顶部显示强度信息
- ⚔️ 进攻 Lv.X 态度词（带颜色编码）
- 🛡️ 防守 Lv.X 态度词（带颜色编码）

**控制台详细输出**
- 输出强度评估的完整过程
- 显示每个子维度的得分（0-10 分）
- 显示总分（0-100 分）和等级

#### 技术实现

**新增文件**
- `src/analysis/strength-evaluator.js`: 强度评估器（完全配置化）
  - `evaluateAttackStrength()`: 评估进攻强度
  - `evaluateDefenseStrength()`: 评估防守强度
  - `generateAttackAttitude()`: 生成进攻态度词
  - `generateDefenseAttitude()`: 生成防守态度词
  - `getStrengthLevel()`: 获取强度等级

**修改文件**
- `src/config/analysis-config.json`: 新增 `strength_calculation` 配置节
  - `attack.weights`: 进攻子维度权重
  - `attack.normalization`: 归一化范围
  - `attack.sub_weights`: 子维度内部权重
  - `attack.attitude_thresholds`: 态度词触发阈值
  - `defense.weights`: 防守子维度权重
  - `defense.normalization`: 归一化范围
  - `defense.sub_weights`: 子维度内部权重
  - `defense.attitude_thresholds`: 态度词触发阈值
- `src/analysis/advice-generator.js`: 集成强度评估
- `src/ui/player-info-card.js`: 显示强度信息
- `src/game/player-processor.js`: 传递强度信息，输出详细日志

#### 配置示例

```json
"strength_calculation": {
  "attack": {
    "weights": {
      "立直进攻": 0.30,
      "副露进攻": 0.25,
      "速度": 0.25,
      "打点": 0.20
    },
    "normalization": {
      "立直率": { "min": 10, "max": 30 },
      "立直收支": { "min": -1000, "max": 3000 },
      "和了巡数": { "min": 14, "max": 9 }
    },
    "attitude_thresholds": {
      "立直狂": { "立直率": 25, "立直进攻": 7 }
    }
  }
}
```

## [2.0] - 2026-03-11

### 正式版发布

**最终可用版本**

#### 新增功能

- ✅ 实时风格分析（9种主称号 + 11种特征标签）
- ✅ 半透明悬浮窗显示（自适应宽度，60%透明度）
- ✅ 颜色编码偏差值（红=高于平均，绿=低于平均，深浅表示程度）
- ✅ 无数据玩家提示（显示"无牌谱数据"）
- ✅ 智能位置分配（自己固定左下角，其他玩家动态分配）
- ✅ 退出房间自动清除信息框
- ✅ 快速加载（2秒启动，1秒检测间隔）

#### 技术特点

- 使用 `unsafeWindow` 访问游戏对象
- 完全合规，不触碰雀魂反爬机制
- 仅调用牌谱屋公开 API
- ES5 兼容，无外部依赖

## 版本号规范

本项目采用语义化版本号（Semantic Versioning）：`vMAJOR.MINOR.PATCH`

- **MAJOR（主版本号）**：不兼容的API修改
- **MINOR（次版本号）**：向下兼容的功能性新增
- **PATCH（修订号）**：向下兼容的问题修正
