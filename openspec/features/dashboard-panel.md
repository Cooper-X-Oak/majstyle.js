# 功能规范：交互式数据分析面板

## 概述

在雀魂游戏页面内创建一个可拖拽的悬浮面板，提供玩家数据查询、可视化分析和多玩家对比功能。

## 用户故事

作为雀魂玩家，我希望能够：
1. 快速搜索任意玩家的历史数据
2. 通过图表直观对比多个玩家的风格差异
3. 在不离开游戏的情况下进行数据分析

## 功能需求

### 1. 面板基础功能

#### 1.1 触发方式
- 添加快捷键：`Ctrl + Shift + D` 打开/关闭面板
- 添加悬浮按钮：屏幕右下角固定按钮

#### 1.2 面板特性
- 可拖拽移动位置
- 可调整大小（最小 600x400px，最大 1200x800px）
- 可最小化/最大化
- 半透明背景（可调节透明度）
- 记住用户的位置和大小偏好（localStorage）

### 2. 玩家搜索功能

#### 2.1 搜索方式
- 输入框支持：
  - 玩家 ID（数字）
  - 玩家昵称（模糊搜索）
  - 从当前对局快速添加（一键添加所有对手）

#### 2.2 搜索结果
- 显示玩家基本信息：
  - 昵称、ID、段位
  - 总对局数、最近活跃时间
- 显示风格标签和称号
- 添加到对比列表按钮

### 3. 数据可视化

#### 3.1 雷达图（主要）
- 6 维度雷达图：
  - 立直率
  - 副露率
  - 和牌率
  - 放铳率（反向）
  - 平均打点
  - 进攻意愿
- 支持叠加显示 2-4 个玩家
- 不同颜色区分玩家
- 显示段位基准线（虚线）

#### 3.2 柱状对比图
- 横向对比多个玩家的关键指标
- 颜色编码：红色=高于平均，绿色=低于平均

#### 3.3 趋势折线图（未来扩展）
- 显示玩家近期数据变化趋势
- 需要 API 支持时间序列数据

### 4. 多玩家对比

#### 4.1 对比列表
- 最多同时对比 4 个玩家
- 列表显示：
  - 玩家昵称
  - 风格称号
  - 删除按钮

#### 4.2 对比视图
- 并排显示详细数据
- 高亮差异最大的指标
- 生成对局建议

## 技术实现

### UI 框架
- 纯 JavaScript（ES5 兼容）
- 无外部 UI 框架依赖

### 图表库
- Chart.js v3.x（CDN 引入）
- 轻量级，支持雷达图、柱状图、折线图
- 响应式设计

### 数据存储
- localStorage 存储：
  - 用户偏好设置
  - 最近搜索历史
  - 收藏的玩家列表

### API 集成
- 复用现有的牌谱屋 API 调用逻辑
- 添加请求缓存（5 分钟有效期）
- 批量请求优化

## UI 设计

### 面板布局

```
┌─────────────────────────────────────────┐
│ 🎯 雀魂数据分析面板          [_] [□] [×] │
├─────────────────────────────────────────┤
│ 搜索: [___________] [搜索] [添加当前对局] │
├─────────────────────────────────────────┤
│ 对比列表 (2/4):                          │
│ ☑ 玩家A [狂战士] [×]                     │
│ ☑ 玩家B [忍者]   [×]                     │
├─────────────────────────────────────────┤
│ [雷达图] [柱状图] [详细数据]             │
│                                          │
│        📊 图表显示区域                   │
│                                          │
│                                          │
└─────────────────────────────────────────┘
```

### 配色方案
- 背景：rgba(20, 20, 30, 0.95)
- 主色：#ffd700（金色）
- 强调色：#ff6b6b（红）、#51cf66（绿）
- 文字：#ffffff

## 性能要求

- 面板打开时间：< 200ms
- 搜索响应时间：< 500ms
- 图表渲染时间：< 300ms
- 内存占用：< 20MB

## 兼容性

- Chrome/Edge 90+
- Firefox 88+
- 与现有脚本功能不冲突

## 开发优先级

### P0（必须）
- 面板基础框架
- 玩家搜索
- 雷达图可视化
- 多玩家对比

### P1（重要）
- 柱状对比图
- 数据缓存
- 用户偏好保存

### P2（可选）
- 趋势折线图
- 收藏功能
- 导出报告

## 测试用例

1. 打开/关闭面板
2. 搜索已知玩家
3. 添加当前对局玩家
4. 对比 2-4 个玩家
5. 切换图表类型
6. 拖拽和调整大小
7. 刷新页面后恢复位置

- 添加数据导出功能（CSV/JSON）
