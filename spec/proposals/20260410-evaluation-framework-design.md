**状态**: 🟡 设计探索中

## 背景与目标

当前称号体系用「进攻意愿（频率）× 放铳率（频率）」两个轴，描述的是**行为频率**，不是**行为质量**。

用户提出的核心洞察：**高手是极化攻防 + 注重顺位**。这意味着评价体系需要能区分：
- 打得多 vs 打得好
- 守得少 vs 守得稳
- 技术强 vs 意识强

本文档不预设结论，而是通过**控制变量穷举推理**，从 API 的核心指标出发，探索每个指标组合背后真正对应的玩家类型，为后续称号设计提供原材料。

---

## 方法论

固定其他变量，只改变一个指标，推理「这个人是什么样的人」。

每个推理单元的格式：
```
固定：A 指标 = X
变量：B 指标 高 / 低
→ 推论：这个人在做什么？为什么？
→ 玩家类型标签
```

---

## 一、净打点效率轴

`净打点效率 = 打点效率 - 铳点损失`

这是最综合的结果指标，先以它为基准做控制变量。

---

### 1.1 净打点效率高 × 平均顺位好

```
净打点效率 > 800，平均顺位 < 2.4

推论：
  每局净赚多，且排名靠前。
  说明这个人不只是「打得好」，还懂得把技术转化为顺位。
  在该放弃时放弃（不追求打点而保顺位），在该抢一位时抢一位。

玩家类型：全能型高手 / 棋手
特征词：技术 + 意识双强，极化攻防，顺位博弈
```

---

### 1.2 净打点效率高 × 平均顺位差

```
净打点效率 > 800，平均顺位 > 2.6

推论：
  每局净赚多，但排名不好。
  说明这个人技术很强，但打法不考虑顺位——
  可能在已经三位的情况下还在追求打点，
  或者不会在末位时做高风险博弈。
  「只管出剑，不管场面」。

玩家类型：技术派 / 剑客
特征词：打点强，顺位意识弱，自我风格强
```

---

### 1.3 净打点效率低 × 平均顺位好

```
净打点效率 < 200，平均顺位 < 2.5

推论：
  每局净赚少，但排名不错。
  说明这个人技术一般，但顺位意识强——
  懂得在合适的时机放弃打点换顺位，
  或者擅长利用场面优势（如一位时极度保守）。
  「算盘打得精，但底子薄」。

玩家类型：顺位博弈型 / 算盘
特征词：技术中等，顺位意识强，场面感好
```

---

### 1.4 净打点效率低 × 平均顺位差

```
净打点效率 < 200，平均顺位 > 2.7

推论：
  每局净赚少，排名也差。
  技术和意识都没有突出之处。
  可能是新手，也可能是「随便打打」的休闲玩家。

玩家类型：普通玩家 / 散漫
特征词：无明显特征，随机性强
```

## 二、进攻质量轴

固定「进攻频率中等」，变化进攻质量指标。

---

### 2.1 立直收支高 × 立直率高

```
立直收支 > 3000，立直率 > 22%

推论：
  立直多，且立直盈利。
  说明立直时机判断准确，好型多，先制率高。
  立直是真正的武器，不是在赌。

玩家类型：立直型高手
特征词：立直精准，先制意识强，立直收支正
```

---

### 2.2 立直收支低 × 立直率高

```
立直收支 < 0，立直率 > 22%

推论：
  立直多，但立直亏损。
  说明立直时机差——可能振听立直多，或者被追后还不撤退，
  或者好型率低（赌博立直）。
  「立直狂但立直烂」。

玩家类型：赌博立直型
特征词：立直频繁，质量差，容易被对攻打中
```

---

### 2.3 副露后和牌率高 × 副露率高

```
副露后和牌率 > 35%，副露率 > 35%

推论：
  副露多，且副露后能收尾。
  说明副露判断准确，速攻能力强，
  鸣牌后不会陷入困境。

玩家类型：副露型高手
特征词：速攻流，副露效率高，节奏快
```

---

### 2.4 副露后和牌率低 × 副露率高

```
副露后和牌率 < 25%，副露率 > 35%

推论：
  副露多，但副露后收不了尾。
  说明副露判断差——可能鸣牌后手牌变差，
  或者被对手立直后无法撤退，
  或者只是「看到牌就鸣」。

玩家类型：乱副露型
特征词：副露频繁，效率低，容易被立直打中
```

---

### 2.5 默听率高 × 放铳率低

```
默听率 > 15%，放铳率 < 13%

推论：
  经常不立直不副露直接和牌，且放铳少。
  说明这个人擅长隐蔽听牌，进攻意图不明显，
  防守意识也强（不会因为默听而放松警惕）。
  「最难读的类型」。

玩家类型：默听猎手
特征词：进攻隐蔽，防守稳健，难以预判
```

---

### 2.6 默听率高 × 放铳率高

```
默听率 > 15%，放铳率 > 16%

推论：
  经常默听，但也经常放铳。
  说明这个人喜欢追求打点（默听大牌），
  但防守意识差，容易在追求打点时放铳。
  「贪心型」。

玩家类型：贪心型
特征词：追求大牌，防守差，高风险高收益
```

## 三、防守质量轴

固定「进攻频率中等」，变化防守质量指标。

---

### 3.1 放铳率低 × 平均铳点低

```
放铳率 < 13%，平均铳点 < 5000

推论：
  放铳少，放了也是小牌。
  这是真正的极化防守——不只是「不放铳」，
  而是即使放铳也能控制损失。
  说明防守时能准确判断危险牌，
  宁可放小牌也不放大牌。

玩家类型：铁壁型
特征词：防守极化，损失控制强，大牌防守好
```

---

### 3.2 放铳率低 × 平均铳点高

```
放铳率 < 13%，平均铳点 > 6000

推论：
  放铳少，但一放就是大牌。
  说明这个人防守策略是「尽量不放铳」，
  但一旦判断失误就损失惨重。
  可能是「过度自信型」——觉得自己不会放铳，
  结果放了就是满贯。

玩家类型：脆弱防守型
特征词：放铳率低但被炸率高，防守有盲点
```

---

### 3.3 放铳率高 × 平均铳点低

```
放铳率 > 16%，平均铳点 < 4800

推论：
  放铳多，但放的都是小牌。
  说明这个人防守策略是「宁可放小牌，不要流局」，
  主动用小牌放铳换取进攻机会。
  可能是速攻流的副产品——副露多，容易被打中，但损失可控。

玩家类型：主动放铳型
特征词：放铳多但损失小，进攻优先，防守次之
```

---

### 3.4 放铳率高 × 平均铳点高

```
放铳率 > 16%，平均铳点 > 6000

推论：
  放铳多，且放的都是大牌。
  防守能力全面薄弱——既不能减少放铳次数，
  也不能控制放铳损失。
  「漏勺」的极端形态。

玩家类型：漏勺型
特征词：防守全面薄弱，容易成为他人的分数来源
```

## 四、速度轴

---

### 4.1 和了巡数快 × 副露率高

```
和了巡数 < 11，副露率 > 35%

推论：
  速度快，靠副露实现。
  典型的速攻流——通过鸣牌快速完成听牌，
  抢在对手立直前和牌。
  对手立直后压力大。

玩家类型：速攻流
特征词：节奏快，副露主导，抢先和牌
```

---

### 4.2 和了巡数快 × 立直率高

```
和了巡数 < 11，立直率 > 22%

推论：
  速度快，靠立直实现。
  说明起手向听好，或者追求速立直（不等好型）。
  立直巡目可能很早（< 8 巡）。

玩家类型：早立直型
特征词：立直早，节奏快，先制压力大
```

---

### 4.3 和了巡数慢 × 平均打点高

```
和了巡数 > 13，平均打点 > 7500

推论：
  速度慢，但打点高。
  说明这个人在等大牌——不急于和牌，
  宁可慢一点也要做出高打点的手牌。
  「价值型」玩家。

玩家类型：价值型
特征词：追求大牌，速度慢，打点高，一旦和牌损失大
```

---

### 4.4 和了巡数慢 × 平均打点低

```
和了巡数 > 13，平均打点 < 6000

推论：
  速度慢，打点也低。
  说明这个人既不追求速度也不追求打点，
  可能是防守优先——宁可流局也不冒险进攻。
  或者是手牌运气差，经常起手向听多。

玩家类型：保守型 / 乌龟
特征词：进攻意愿低，流局率高，防守优先
```

## 五、立直质量深度轴

---

### 5.1 先制率高 × 被追率低

```
先制率 > 85%，被追率 < 15%

推论：
  立直时几乎都是先手，且很少被追。
  说明立直时机好——在对手还没准备好时就立直，
  且立直后对手不敢追（可能是好型，对手知道追不过）。

玩家类型：先制王
特征词：立直时机精准，先制优势强，对手压力大
```

---

### 5.2 追立率高 × 立直收支正

```
追立率 > 25%，立直收支 > 2000

推论：
  经常追立直，且追立直盈利。
  说明这个人对攻能力强——能判断什么时候值得追，
  且追了之后能赢。
  「对攻狂魔」的高质量版本。

玩家类型：对攻型高手
特征词：追立直精准，对攻能力强，不怕正面交锋
```

---

### 5.3 追立率高 × 立直收支负

```
追立率 > 25%，立直收支 < 0

推论：
  经常追立直，但追了亏损。
  说明这个人喜欢对攻，但判断差——
  在不该追的时候追，结果被对手打中。
  「莽夫型」。

玩家类型：莽夫型
特征词：好斗，对攻判断差，容易被高质量立直打中
```

## 六、综合推理：极化攻防的特征组合

用户说的「高手是极化攻防」，对应的指标组合是什么？

```
极化进攻的特征：
  - 净打点效率高（进攻产出 >> 防守代价）
  - 立直收支正（立直是武器）
  - 平均打点高（赢了赢大）
  - 先制率高（立直时机好）

极化防守的特征：
  - 放铳率低（少放铳）
  - 平均铳点低（放了也是小牌）
  - 被炸率低（不放大牌）
  - 立直后放铳率低（立直后也能撤退）

两者同时具备 → 真正的高手
只有进攻极化 → 攻击型（可能防守有漏洞）
只有防守极化 → 防守型（进攻可能不够锋利）
两者都没有   → 普通玩家
```

## 七、待探索的维度

以下组合还没有充分推理，留待后续：

1. **流局率高 × 流听率高** — 是保守还是手牌差？
2. **一发率高 × 立直巡目早** — 速立直的质量如何？
3. **里宝率高** — 运气成分 vs 立直质量的关系
4. **放铳时立直率高** — 被对攻打中 vs 主动对攻失败
5. **平均被炸点数高** — 被炸时损失大，说明什么？

## 八、初步轴线候选

基于以上推理，浮现出以下几个有区分度的轴线候选：

| 轴线 | 正极 | 负极 | 核心指标 |
|------|------|------|---------|
| 实力轴 | 净打点效率高 | 净打点效率低 | 净打点效率 |
| 顺位意识轴 | 顺位好于预期 | 顺位差于预期 | 平均顺位 vs 净打点效率 |
| 进攻方式轴 | 立直主导 | 副露主导 | 立直率 vs 副露率 vs 默听率 |
| 进攻质量轴 | 立直收支正 + 副露后和牌率高 | 立直收支负 + 副露后和牌率低 | 立直收支、副露后和牌率 |
| 防守质量轴 | 放铳率低 + 平均铳点低 | 放铳率高 + 平均铳点高 | 放铳率 × 平均铳点 |
| 速度轴 | 和了巡数快 | 和了巡数慢 | 和了巡数 |
| 隐蔽性轴 | 默听率高 + 放铳率低 | 默听率低 | 默听率 |

**关键问题**：哪几个轴线组合起来，能最有效地区分玩家类型，且对对局有实际指导意义？
