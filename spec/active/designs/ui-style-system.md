## 概述

本文档固化 majstyle.js 当前 UI 的视觉风格，作为后续所有 UI 改动的基准参考。
目标是让任何人（包括 AI agent）在修改 UI 时都能保持风格一致，不需要反复读源码。

**v1.1 变更**: 所有设计值已从组件代码中抽离，集中到 `src/ui/design-tokens.js`。
修改视觉风格只需编辑该文件，无需触碰组件逻辑。

---

## 视觉语言

### 整体基调

- 暗色半透明浮层，叠加在雀魂游戏画面上
- 不干扰游戏操作，默认 `pointer-events: none`（增强版卡片除外，支持点击展开）
- 极简信息密度：精简视图只显示最关键的 3 个数据点

### 容器样式（基准）

```
background: rgba(0,0,0,0.75)
color: #fff
padding: 6px 10px
border-radius: 8px
font-size: 10px
z-index: 10000
box-shadow: 0 0 10px rgba(0,0,0,0.5)
max-width: 200px（精简）/ 280px（展开）
transition: all 0.2s ease
```

无数据状态用更浅的背景 `rgba(0,0,0,0.5)` 和灰色文字 `#888`，视觉上降权。

## 布局规则

### 四角定位系统

四人麻将四个玩家卡片固定在屏幕四角，通过 `position: fixed` 实现：

| 玩家 | 位置 | CSS |
|------|------|-----|
| 自己 | 左下角 | `bottom: 140px; left: 10px` |
| 对手 1 | 右上角 | `top: 140px; right: 10px` |
| 对手 2 | 右上角（更靠上） | `top: 10px; right: 10px` |
| 对手 3 | 左上角 | `top: 140px; left: 10px` |

对手位置通过 `window.majstyleJS.playerUICounter` 全局计数器顺序分配，每局重置。

## 颜色系统

### 偏差颜色编码（`color-utils.js`）

用于表示玩家数据相对于段位基准的偏差方向和强度：

| 偏差方向 | 强度 | 颜色 |
|----------|------|------|
| 正偏差（高于基准） | 强（>2x threshold） | `#ff4444` 红 |
| 正偏差 | 中（>threshold） | `#ff6b6b` 浅红 |
| 正偏差 | 弱 | `#ff9999` 淡红 |
| 负偏差（低于基准） | 强 | `#44ff44` 绿 |
| 负偏差 | 中 | `#51cf66` 浅绿 |
| 负偏差 | 弱 | `#99ff99` 淡绿 |
| 无偏差 | — | `#aaa` 灰 |

**语义约定**：红色 = 高于均值（不一定是"危险"），绿色 = 低于均值。
具体含义取决于指标：放铳率高（红）是危险信号，立直率高（红）是激进信号。

### 危险度颜色（`getDangerColor`）

| 分数 | 颜色 | 含义 |
|------|------|------|
| 9-10 | `#ff1744` | 极危险 |
| 7-8 | `#ff9800` | 高危 |
| 5-6 | `#ffc107` | 中等 |
| 3-4 | `#8bc34a` | 较低 |
| 0-2 | `#4caf50` | 安全 |

### 优先级颜色（`getPriorityColor`，用于策略建议左边框）

| 优先级 | 颜色 |
|--------|------|
| ≥8 | `#ff1744` |
| ≥6 | `#ff9800` |
| ≥4 | `#ffc107` |
| ≥2 | `#8bc34a` |
| <2 | `#4caf50` |

### 固定语义色

| 用途 | 颜色 |
|------|------|
| 原型标题 / 称号 | `#ffd700` 金色 |
| 玩家昵称 | `#aaa` 灰色 |
| 标签文字 | `#ffa500` 橙色 |
| 策略建议标题 | `#4fc3f7` 浅蓝 |
| 次要说明文字 | `#999` |
| 分隔线 | `rgba(255,255,255,0.2)` |
| 折叠提示文字 | `#666` |

## 字体规格

| 层级 | 大小 | 用途 |
|------|------|------|
| 标题 | 11px bold | 原型称号 |
| 正文 | 9-10px | 数据行、强度标签 |
| 辅助 | 8-9px | 策略建议、折叠提示 |

行高：`line-height: 1.4`

---

## 交互规则

- 默认精简视图，`pointer-events: auto`，`cursor: pointer`
- 点击切换精简/展开，`max-width` 同步变化（200px → 280px）
- 点击事件 `stopPropagation`，不穿透到游戏层
- 无数据卡片 `pointer-events: none`，不可交互
