## 🚀 立即开始

### 推荐方式: 使用游戏内探索模式（10 分钟）

由于浏览器 CORS 限制，独立的 HTML 工具无法直接访问 API。**推荐直接使用游戏内探索模式**，它可以同时探索游戏对象和 API 数据。

#### 步骤 1: 安装更新后的脚本

1. **找到脚本文件**
   ```
   文件: dist/雀魂金玉四麻风格分析助手-v2.0.0.user.js
   ```

2. **在 Tampermonkey 中安装**
   - 如果已安装旧版本，先删除或更新
   - 拖拽脚本文件到浏览器，或在 Tampermonkey 中导入

#### 步骤 2: 进入雀魂对局

1. **打开雀魂游戏**
   - 访问雀魂官网或客户端

2. **进入对局**
   - 必须在对局中（四人麻将）
   - 不能在大厅或其他界面

#### 步骤 3: 触发探索模式

1. **按下快捷键**
   ```
   Ctrl + Shift + D
   ```

2. **打开浏览器控制台**
   ```
   按 F12 键
   或右键 -> 检查 -> Console 标签
   ```

3. **查看输出**
   - 控制台会输出大量信息
   - 包括游戏对象结构
   - 包括完整的 API 数据（JSON 格式）

#### 步骤 4: 记录结果

1. **复制游戏对象结构**
   - 找到 "=== 雀魂游戏对象结构探索 ===" 部分
   - 复制所有属性列表
   - 粘贴到 `docs/GAME_OBJECT_STRUCTURE.md`

2. **复制 API 数据**
   - 找到 "【player_extended_stats 完整响应】" 部分
   - 复制完整的 JSON 数据
   - 粘贴到 `docs/API_DATA_STRUCTURE.md`

3. **复制字段清单**
   - 找到 "【字段清单】" 部分
   - 记录所有字段名和类型
   - 更新到文档中

### 备用方式: API Explorer HTML 工具

如果你想使用独立的 HTML 工具，需要：

1. **启动本地 Web 服务器**
   ```bash
   cd tools
   python -m http.server 8000
   ```

2. **在浏览器中访问**
   ```
   http://localhost:8000/api-explorer.html
   ```

3. **输入账号 ID 并探索**

**注意**: 这种方式仍然可能遇到 CORS 问题，推荐使用游戏内探索模式。

### 步骤 5: 分析数据（10 分钟）

1. **查看 API 返回的所有字段**
   - 哪些字段是我们已经在使用的？
   - 哪些字段是新发现的？
   - 哪些字段可能有价值？

2. **评估字段价值**
   - 高价值：直接影响对局策略
   - 中价值：辅助分析
   - 低价值：纯展示性数据

3. **更新文档**
   - 在 `docs/API_DATA_STRUCTURE.md` 中填写完整的字段清单
   - 标记每个字段的价值评估

## ❓ 常见问题

**Q: API Explorer HTML 工具显示 "Failed to fetch" 错误？**

A: 这是浏览器的 CORS（跨域资源共享）限制导致的。本地 HTML 文件无法直接访问外部 API。
解决方案：
1. **推荐**: 使用游戏内探索模式（`Ctrl+Alt+E`），它使用 Tampermonkey 的特权可以绕过 CORS
2. 启动本地 Web 服务器：`python -m http.server 8000`，然后访问 `http://localhost:8000/api-explorer.html`
3. 直接使用游戏内探索模式是最简单的方法

**Q: 我不知道我的账号 ID 是什么？**

A: 有几种方法获取：
1. 在雀魂游戏中进入对局，按 `Ctrl+Alt+E`，控制台会显示
2. 在牌谱屋网站搜索你的昵称，URL 中包含账号 ID
3. 使用浏览器开发者工具查看游戏的网络请求

**Q: 游戏内探索模式没有输出？**

A:
1. 确保在对局中（不是大厅）
2. 确保脚本已正确安装和启用
3. 打开浏览器控制台（F12）查看
4. 检查控制台是否有错误信息
5. 尝试刷新页面重新进入对局

**Q: 完成探索后做什么？**

A:
1. 将结果记录到文档中
2. 分析数据并评估价值
3. 基于实际数据更新 `docs/DATA_ANALYSIS_IDEAS.md`
4. 准备开始 Phase 1: 建议系统增强
